# Tool Guides — Coach the User Through Free Tools

These are the step-by-step scripts to hand the user, one tool at a time. App menus change over
time, so **adapt to what the user reports seeing** — if a button isn't where this says, ask them
what they see and adjust. Always give numbered, bolded steps and confirm each tool worked before
moving on. All of these tools are free.

---

## Getting a transcript
- **YouTube:** under the video, **More (...)** → **Show transcript** → select all → copy → paste into the chat.
- **Audio/video file only:** the simplest path is to upload the file directly into the Claude chat and work from that. If the file is too big, suggest one free transcription site and keep it to that single recommendation.
- Keep it to ONE recommended path for their situation. Don't hand them a menu of options.

---

## CapCut — cut one clip (free, no account needed to start)
Give the exact start/end timestamps from the workflow Step 2.
1. **Open CapCut** (free app on phone or computer, or capcut.com in a browser).
2. **New project** → **Import** the original video file.
3. Drag the video onto the timeline.
4. **Trim to the moment:** move the playhead to the **start timestamp**, split (the scissors), then move to the **end timestamp**, split again. Delete the parts before and after so only the clip remains.
5. **(Optional) captions:** CapCut can auto-add captions — **Captions → Auto captions**. Great for sound-off viewers.
6. **Copyright check:** only add music if it's royalty-free or from the app's licensed library. Do not add copyrighted songs to a clip you'll post.
7. **Export** (top right) → keep 1080p → save the file.
8. Confirm with them: "Did it export okay?" before moving to scheduling.

## Canva — make the quote card (only if they want a graphic)
Hand them the quote text + attribution from the workflow.
1. **Open canva.com** (free account) → search a template size: **Instagram Post (Square)** or **Facebook Post**.
2. Pick a clean template, or use their saved brand template if they have one.
3. **Replace the text** with the quote text I gave you, and the attribution line.
4. Apply their **brand colors** (from the profile) if easy.
5. **Download** → PNG.
6. Tip to relay: **if they have a real photo from the event, that usually beats a graphic** — offer to write the caption for the photo instead.
- If they have no Canva account, this whole thing is the setup — it takes ~2 minutes and is free.

---

## Meta Business Suite — schedule the post (free, no developer API)
This is the normal, no-code way to schedule Facebook + Instagram posts.
1. **Go to business.facebook.com** (or the Meta Business Suite app) and sign in with the account that manages the Page.
2. Left menu → **Content** (or **Planner**) → **Create post**.
3. **Pick the Page** (and Instagram account, if they want both).
4. **Paste the caption** I wrote. **Upload the clip or the graphic.**
5. Click the **schedule** option (a small calendar / "Schedule" toggle), **set the date and time.**
6. **Schedule** (the button may say "Schedule" instead of "Publish").
7. Confirm with them: "Do you see it in your scheduled/planned posts?"
- **Other platforms:** YouTube has its own "Schedule" on upload; TikTok has a scheduler in its web uploader. If they post elsewhere, walk them through that platform's native scheduler the same way — numbered steps, confirm it landed.

---

## General coaching rules
- One tool at a time. Confirm success before the next step.
- Name the exact button label where you can; if the user can't find it, ask what's on their screen.
- Never tell them to install anything heavy, edit settings files, or use a terminal. If a path needs that, find an easier one.
