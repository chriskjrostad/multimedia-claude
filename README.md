# Multimedia Claude

An open **Claude setup** that turns one piece of long-form media — a podcast, webinar, keynote, long video, interview, or recorded talk — into a week of social content, and coaches you click-by-click through finishing and scheduling it in free tools you already use.

- **No code. No terminal. No server.** It runs as a Claude Project in the Claude app.
- Claude is the **director** (picks the best moments, writes the copy, designs the quote card, gives you exact steps). You are the **hands** (a few clicks in tools like CapCut and Meta Business Suite).
- Built for **non-technical content and social-media managers**, not developers.

Same "published Claude configuration" idea as the developer-focused Claude Council — different audience.
